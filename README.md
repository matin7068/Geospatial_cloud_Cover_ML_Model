# Geospatial_cloud_Cover_ML_Model
A deep learning capstone project evaluating U-Net, DeepLabV3+ (Xception41), and SegFormer (mit-b2) for high-resolution cloud segmentation over Guwahati using 4-band Sentinel-2 imagery. Fully integrated with solar irradiance attenuation profiling, solar power yield forecasting, clearness/variability indices, and model prediction uncertainty heatmaps
